Student Data Analysis
Overview

This project analyzes student academic data to understand performance patterns across different subjects. The goal is to identify trends in grades and see how various factors relate to overall academic outcomes.

Dataset

The dataset contains student-level academic information, including:

Student ID

Class

Gender and race

GPA

Subject scores (Algebra, Calculus I & II, Statistics, Probability, Measure, Functional Analysis)

Additional academic indicators (from1 to from4)

Target variable (y)

Tools Used

Python

Jupyter Notebook

Pandas, NumPy

Matplotlib

What I Analyzed

Distribution of GPA and subject scores

Performance comparison across subjects

Relationship between GPA and individual subjects

Overall academic performance patterns

Key Observations

GPA is strongly influenced by performance in core mathematics subjects

Some subjects show higher variability in scores than others

Consistent performance across subjects is linked to better overall outcomes

Conclusion

This analysis provides insights into student performance and highlights how subject-wise results contribute to overall academic success.
